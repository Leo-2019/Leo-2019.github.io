### 滑板的桥（truck） <a id="桥"></a>
桥分硬桥和软桥。滑板桥的 “硬” 和 “软” 核心取决于 PU（聚氨酯衬套）的硬度（其次是桥的结构设计），直接影响转向灵活性、稳定性、脚感反馈和适用场景，本质是 “操控灵敏度” 与 “支撑稳定性” 的权衡。

<p style="
  text-align: center; /* 居中 */
  color: #d32f2f; /* 颜色（深红色） */
  font-weight: bold; /* 加粗 */
  font-family: 'STKaiti', 'KaiTi', cursive; /* 字体（楷体优先） */
">
  硬桥 =“稳定 + 直接”：适配新手、街式动作、高速滑行，核心是 “好掌控、不晃”;
</p>

<p style="
  text-align: center; /* 居中 */
  color: #d32f2f; /* 颜色（深红色） */
  font-weight: bold; /* 加粗 */
  font-family: 'STKaiti', 'KaiTi', cursive; /* 字体（楷体优先） */
">
  软桥 =“灵活 + 缓冲”：适配碗池、高落差动作、巡航，核心是 “好转向、吸震”。
</p>

如果玩法是 “街式为主，偶尔玩碗池”，优先硬桥；如果 “碗池为主，街头为辅”，优先软桥；新手闭眼选硬桥（95A 左右 PU）。

**为什么 PU（bushings） 硬度是核心？**

<img class="img-center" align="center" src="https://leo-2019-blog-1259040785.cos.ap-beijing.myqcloud.com/Image/20251204204844513.webp" width="50%" alt="按宽度比例缩放">

滑板桥的 bushings（PU） 是桥身与底座之间的 “缓冲垫”，分上下两块（上 PU 控制转向幅度，下 PU 负责支撑），硬度用 邵氏硬度（A） 标注：
{% note success%} 
**硬 PU（95A+）：分子密度高，形变能力弱 → 转向时阻力大，所以 “硬”，但支撑力强，不会轻易晃动；**
**软 PU（85A-92A）：分子密度低，易形变 → 转向时阻力小，所以 “软”，但支撑力弱，受力时会轻微变形缓冲。**
{% endnote %}
部分品牌会通过 PU 的形状（锥形 / 圆柱形）、桥的结构（矮桥 / 高桥）辅助调整 “软硬感”，但核心还是 PU 硬度。

### 轴承（bearing） <a id="轴承"></a>

作为滑板中不可或缺的部件，轴承承载着连接轮子和支架的作用，它的存在让轮子能转得更顺畅，一副好的轴承配上合适的轮子，能让滑板的滑行速度更快、续航能力更强。
