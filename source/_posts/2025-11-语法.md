> 一些复杂的Mardown 语法。
<!--more-->

### 居中引用
效果：

{% cq %} 引用文字 {% endcq %}

{% codeblock%}
{% cq %} 引用文字 {% endcq %}
{% endcodeblock %}



## 彩色 Tags

效果：

{% label default @ default %} 
{% label primary @ primary %} 
{% label success @ success %} 
{% label info @ info %} 
{% label warning @ warning %} 
{% label danger @ danger %} 
{% codeblock%}
{% label [class] @ 标签内容 %}

class：
 - default: 浅灰色,可留空
 - primary: 浅紫色
 - success: 浅绿色
 - info: 浅蓝色
 - warning: 浅黄色
 - danger: 浅红色
{% endcodeblock %}





### 彩色 Note

效果：

{% note success%} 
内容
{% endnote %}

{% codeblock %}
{% note [class]%} 
内容
{% endnote %}

class: 
 - default: 灰色
 - primary: 紫色
 - success: 绿色
 - info: 蓝色
 - warning: 黄色
 - danger: 红色
{% endcodeblock %}

## 切换Tabs

效果：

{% tabs [Class] %}

<!-- tab 1号Tab-->
这里是1号Tab内容
<!-- endtab -->

<!-- tab 2号Tab-->
这里是2号Tab内容
<!-- endtab -->

<!-- tab 3号Tab-->
这里是3号Tab内容
<!-- endtab -->

{% endtabs %}

{% codeblock %}

{% tabs [Class] %} // Class: 必填项，Tab的全局名称，唯一值，不能带有符号

<!-- tab 1号Tab-->
这里是1号Tab内容
<!-- endtab -->

<!-- tab 2号Tab-->
这里是2号Tab内容
<!-- endtab -->

<!-- tab 3号Tab-->
这里是3号Tab内容
<!-- endtab -->

{% endtabs %}

{% endcodeblock %}

## 按钮 Btn

{% btn https://example.com , 访问示例网站 , external-link fa-lg %}

{% codeblock %}
{% btn [地址] , [内容] , [图标 规格] %}
[地址]：当使用外链绝对引用时，无按钮，样式类似Markdown链接跳转
[图标 规格]: 使用无`fa-`开头的FontAwesome图标，可以用以下规格定义图标大小
fa-fw | fa-lg | fa-2x | fa-3x | fa-4x | fa-5x
{% endcodeblock %}
